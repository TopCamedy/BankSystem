**BankSystem** — это консольное приложение, имитирующее работу банковского узла. Проект построен на принципах объектно-ориентированного программирования и включает в себя иерархию классов для управления различными типами счетов.

---

## 🛠 Основные возможности

| Функция | Описание |
| :--- | :--- |
| **Регистрация** | Создание учетных записей клиентов в системе. |
| **Управление счетами** | Открытие стандартных и накопительных счетов. |
| **Транзакции** | Пополнение баланса и снятие средств (с проверкой лимитов). |
| **Мультивалютность** | Поддержка RUB, USD, EUR через Enum. |

---

## 🏗 Архитектура проекта

### 📂 Иерархия классов
* `AbstractAccount` — базовый класс, определяющий логику ID и баланса.
* `BankAccount` — стандартная реализация счета.
* `SavingAccount` — специализированный счет с **минимальным остатком 1000 RUB**.
* `Client` — модель данных клиента со списком его счетов.

### ⚙️ Бизнес-логика
Проект демонстрирует использование **полиморфизма**: метод `Withdraw` (снятие) ведет себя по-разному в обычном и накопительном счетах.
